Q 1
Fonction de transfert électrique G_e(p) = I_m(p) / [U_m(p) − E_m(p)]
▼

La loi des mailles dans le circuit d'induit (modèle équivalent : résistance R + inductance L en série avec la f.c.é.m.) donne :

u(t) = R·i(t) + L·di(t)/dt + e(t)

En appliquant la transformée de Laplace avec des conditions initiales nulles :

U(p) − E(p) = R·I(p) + L·p·I(p) = I(p)·(R + L·p)

On en déduit directement la fonction de transfert :

Gₑ(p) = I(p) / [U(p)−E(p)] = 1/(R + L·p) = (1/R) / (1 + (L/R)·p)
Forme canonique du premier ordre : Gₑ(p) = Kₑ / (1 + τₑ·p)
avec Kₑ = 1/R (gain statique électrique) et τₑ = L/R (constante de temps électrique)

Q 3
Fonction de transfert mécanique G_m(p) = Ω(p) / C_m(p)
▼

Le principe fondamental de la dynamique en rotation (équation de Newton) :

C_m(t) − C_r(t) = J·dΩ/dt

Le frottement fluide (visqueux) introduit un couple résistant proportionnel à la vitesse :

C_r(t) = f·Ω(t)

En substituant et en appliquant Laplace :

C_m(p) = J·p·Ω(p) + f·Ω(p) = Ω(p)·(J·p + f)
G_m(p) = Ω(p)/C_m(p) = 1/(J·p + f) = (1/f) / (1 + (J/f)·p)
Forme canonique : G_m(p) = K_mec / (1 + τₘ·p)
avec K_mec = 1/f et τₘ = J/f (constante de temps mécanique dominante)

▼

En combinant les équations électrique, mécanique et électromécanique (C_m = kₘ·I, E = kₘ·Ω), on obtient le schéma bloc complet avec la boucle de contre-réaction par la f.c.é.m. :

La f.c.é.m. Eₘ = kₘ·Ω crée une boucle de contre-réaction interne qui couple les parties électrique et mécanique. Cette boucle est présente même en boucle ouverte au sens automatique.

Q 6
FT simplifiée N(p)/Sr(p) = α₂ / (1 + τ₂·p) avec hypothèse τₑ ≈ 0
▼

Avec L/R = 0, l'induit se réduit à sa résistance R. En résolvant la boucle de contre-réaction interne par la f.c.é.m. :

N(p)/Sr(p) = [Kᵤ · kₘ/R · 1/f · 60/(2π)] ───────────────────────────────────── 1 + (kₘ²)/(R·f) + (J/f)·p

En regroupant les termes constants au dénominateur :

N(p)/Sr(p) = α₂ / (1 + τ₂·p) α₂ = Kᵤ · kₘ · 60 / [2π · (R·f + kₘ²)] τ₂ = J·R / (R·f + kₘ²) ← constante de temps mécanique équivalente
Le système en boucle ouverte est un système du 1er ordre. α₂ est le coefficient de transfert statique (tr/min/V) et τ₂ est la constante de temps dominante mesurable expérimentalement.
Q 7
Comportement statique — Réponse à un échelon d'amplitude A (f ≠ 0)
▼

Pour un échelon Sr(t) = A·u(t), soit Sr(p) = A/p. On applique le théorème de la valeur finale :

N(∞) = lim[p→0] p · [α₂/(1+τ₂·p)] · (A/p) = α₂ · A
La vitesse converge vers une valeur finie N∞ = α₂·A. Le frottement fluide f assure la dissipation d'énergie et garantit un gain statique fini.
En régime permanent, le moteur tourne à vitesse constante proportionnelle à la consigne. L'énergie injectée est dissipée par le frottement fluide (équilibre dynamique).
Q 8
Cas f = 0 — Comportement intégrateur (frottement fluide inhibé)
▼

Sans frottement fluide, l'équation mécanique devient C_m(t) = J·dΩ/dt, et en Laplace :

G_m(p)|f=0 = 1/(J·p) ← intégrateur pur

La FT globale devient :

N(p)/Sr(p)|f=0 = [Kᵤ · kₘ · 60] / [2π · R · J · p] = K_int / p

Réponse à un échelon d'amplitude A :

N(t) = K_int · A · t ← rampe croissante sans limite théorique
⚠ Observation clé : Avec f = 0, le système est un intégrateur pur. Sous un échelon, la vitesse croît linéairement (rampe) jusqu'à saturation électrique du moteur. Le t_r5% n'est pas défini (valeur finale infinie).
Cet essai illustre l'importance du frottement fluide. Un intégrateur en boucle ouverte est difficile à contrôler et instable en boucle fermée sans correction adaptée.

Q 1–2
Fonctions de transfert du PID — Réalisation à amplificateurs opérationnels
▼

Pour un AOP idéal (gain infini, impédance d'entrée infinie), chaque bras est un amplificateur inverseur avec impédance d'entrée Z₁ et de contre-réaction Z₂ :

Sortie/Entrée = −Z₂/Z₁ (convention inverseur)

Action Proportionnelle (P) : Z₁ = R₁, Z₂ = R₂

Ep(S)/E(S) = −R₂/R₁ = −Kp

Action Intégrale (I) : Z₁ = R_i, Z₂ = 1/(C_i·p)

Ei(S)/E(S) = −1/(R_i·C_i·p) = −1/(Ti·p) [Ti = R_i·C_i]

Action Dérivée (D) : Z₁ = 1/(C_d·p), Z₂ = R_d

Ed(S)/E(S) = −R_d·C_d·p = −Td·p [Td = R_d·C_d]

Le sommateur final (inverseur) additionne les trois actions :

R(p) = Kp·[1 + 1/(Ti·p) + Td·p] ← PID idéal (forme série)
Forme standard avec filtre dérivé : R(p) = Kp·[1 + 1/(Ti·p) + Td·p/(1 + Td·p/N)]
N ∈ [10–20] : filtre passe-bas atténuant le bruit haute fréquence sur l'action dérivée
Pour TP2 : Ti → ∞ et Td = 0 → action P seule : R(p) = k (constante). Le coefficient k = K₁·K₂ est réglable par l'utilisateur.
Q 1
FT en boucle fermée G(p) = N(p)/C(p) — Mise sous forme canonique
▼

La FT en boucle ouverte (sans retour) est :

FTBO(p) = k · α₁ / (1 + τ₁·p)

Avec retour unitaire, la FT en boucle fermée est (formule de Black) :

G(p) = FTBO / (1 + FTBO) = [k·α₁/(1+τ₁·p)] / [1 + k·α₁/(1+τ₁·p)] = k·α₁ / (1 + τ₁·p + k·α₁)

On factorise par (1 + k·α₁) au dénominateur :

G(p) = [k·α₁/(1+k·α₁)] / [1 + τ₁/(1+k·α₁)·p]
Forme canonique : G(p) = Kf / (1 + τf·p)

Kf = k·α₁ / (1 + k·α₁) ← gain statique BF
τf = τ₁ / (1 + k·α₁) ← constante de temps BF

Q 4–5
Erreurs statiques — Position (échelon) et traînage (rampe)
▼

Erreur statique de position — Entrée échelon C₀·u(t) [C(p) = C₀/p] :

ε(p) = C(p) / (1 + FTBO(p)) = [C₀/p] / [1 + k·α₁/(1+τ₁·p)] ε∞ = lim[p→0] p·ε(p) = C₀ / (1 + k·α₁)
Erreur de position : ε_pos = C₀ / (1 + k·α₁) — diminue quand K augmente, mais reste non nulle (système de classe 0).

Erreur de traînage — Entrée rampe v·t [C(p) = v/p²] :

ε_train = lim[p→0] p · [v/p²] / (1 + k·α₁/(1+τ₁·p)) = +∞
⚠ Le correcteur P pur ne peut pas suivre une rampe : l'erreur de traînage est infinie. Pour annuler cette erreur, il faut ajouter une action intégrale (correcteur PI ou PID → TP3).
